🚧 還在施工的搶票程式(很懶，用ai寫readme, 有人工校稿過了放心食用)
這是一個基於 Python 與 nodriver 底層架構開發的自動化搶票輔助工具。主打突破前端防禦機制與光速自動填單。

⚠️ 狀態回報： 本專案目前仍在持續施工與優化中。面對各大售票網站時常變動的防禦邏輯，偶爾出現 Bug 或是卡彈屬於正常現象，請斟酌使用並隨時做好手動接管的準備！

✨ 目前支援平台
拓元售票系統 (Tixcraft) ✅ (支援定時待命、關鍵字選區、自動辨識驗證碼)

KKTIX ✅

(遠大 TicketPlus 等其他平台尚在開發調整中...)

🚀 GUI 啟動器使用方法
為了降低操作門檻，本專案提供圖形化介面 (GUI)，你完全無須修改任何一行程式碼，只要填寫表單就能發車。

📂 檔案配置準備
下載並解壓縮後，請確保你的資料夾內擁有以下結構，這四個東西必須放在同一層，缺一不可：

_internal/ (包含辨識模型與環境依賴包的資料夾)

TicketBot_Complete.exe (沒有畫面的核心搶票引擎)

Launcher.exe (圖形化啟動器)

config.py (設定檔)

🎮 操作步驟
開啟控制台：對著 Launcher.exe 點擊兩下開啟圖形介面。

選擇目標平台：在頂部的下拉式選單選擇你要搶的平台 (Tixcraft 或 KKTIX)。

輸入搶票參數：

監控網址：貼上你想搶的活動詳細頁面網址。(拓元需要將detail改成game)
livenation會員預購的話，一樣能用但要動手腳，不告訴你嘿嘿嘿嘿嘿

開賣時間：設定準點開搶的時間 (格式請依照介面提示)，程式會在時間到時自動跳轉。

選位策略：可選擇「關鍵字優先」、「由上而下」或「隨機」。

區域/排除關鍵字：輸入你想買的票價或區域 (如：5800、特區)，並可設定不想買的區域 (如：視線不良)。

預購碼：若為會員或信用卡預購場次，請在此填寫通關密碼。(kktix的這個好像還沒做，因為還沒遇到這情況)

張數：選擇需要搶購的票數。

一鍵啟動：參數確認無誤後，點擊 「🔥 啟動」。GUI 會在背景自動將設定寫入 config.py，並喚醒黑色的終端機與瀏覽器主程式為你衝鋒陷陣。

⚠️ 注意事項與免責聲明
環境要求：本程式會自動喚醒並接管 Chrome 瀏覽器。執行期間請保持網路暢通，且絕對不要隨意手動關閉被程式接管的瀏覽器分頁，以免底層連線崩潰 (Connection Closed)。

防毒軟體誤判：由於程式經過 PyInstaller 打包，且內部包含自動化控制邏輯，部分防毒軟體或 Windows Defender 可能會跳出警告或直接阻擋。若無法執行，請暫時將專案資料夾加入防毒軟體的白名單。

關於圖形驗證碼：程式內建 AI 圖形驗證碼辨識模組，但辨識率受網站字體干擾影響，並非 100%。若遇到辨識失敗，機器人會自動刷新圖片並無限重試，直到成功送出為止。

特殊預購場次 (如 Live Nation)：面對帶有複雜跳轉 Token 的外部預購場次，建議善用介面上的「暫停」功能，手動點擊預購按鈕帶入憑證後，再解除暫停讓機器人接手後續流程。

風險自負：本工具僅供程式語言學習、網站架構研究與自動化交流使用。請勿用於大量掃票之商業黃牛行為。使用本程式所產生之任何購票糾紛或帳號封鎖，開發者概不負責。
